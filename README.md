
# WhistlerLang 🐧✨

Welcome to **WhistlerLang** – the cheeky little programming language that talks to humans, not robots! It's a **medium-level hybrid runtime language** designed to be simple, readable, and a tad mischievous 😏.

---

## Overview 📝

WhistlerLang is for those who want to write code that feels like writing a note to a friend:

- `say("Hello, mate!")` ✅ – no ceremony, just words.
- `math("1 + 2 - 1")` ➗ – simple maths, no stress.
- `concat("Good ", "morning")` 🌞 – string juggling made fun.

It’s **readable**, **logical**, and occasionally **sassy**. Perfect for small scripts, experiments, or just showing off to your mates.

---

## Features 🎩

- **Friendly Syntax** 🐱: Humans understand it, machines obey it.  
- **Hybrid Runtime** ⚡: Fast, safe, and clever enough to handle your chaos.  
- **Built-in Maths** ➕➖: Basic arithmetic, neat and tidy.  
- **String Tricks** ✨: Concatenate and manipulate without headaches.  
- **Interactive REPL** 🎮: Play with code live or run scripts.  
- **Cross-Platform** 🌍: Works on Linux, Android (Termux), and wherever Zig can go.

---

## Usage 🚀

### Run a script:

whistlerlang run <script.whlst>

Example meow.whlst:

say("Meow! 🐱")
math("2 + 3 - 1")
concat("Good ", "morning! 🌞")

./whistlerlang run meow.whlst
# Output:
# Meow! 🐱
# 4
# Good morning! 🌞

Interactive mode:

./whistlerlang
>>> say("Hello, world! 🌎")
>>> math("5 + 7")
>>> concat("Hi ", "there!")
>>> quit


---

Philosophy 🎯

WhistlerLang isn’t about being fancy or overcomplicated.
It’s about making programming feel human, a bit cheeky, and absolutely approachable.
Think of it as coding with a friend who’s slightly British, very polite, and occasionally throws in an emoji
